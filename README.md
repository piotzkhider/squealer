## Quick Start

1. Install via yarn:

```sh
yarn
```

2. Set-up:

serverless.yml

```yml
provider:
  ...
  environment:
    GITHUB_ACCESS_TOKEN: your_access_token123
    GITHUB_WEBHOOK_SECRET: your_webhook_secret456
    SLACK_WEBHOOK_URL: https://hooks.slack.com/services/XXXXXXXX/XXXXXXXXXXXX
```

src/builders/public.ts

```ts
      channel: 'SLACKCHANNELID',
```

3. Deploy a Service:

```sh
sls deploy -v
```
